# API Products
## Descripción

Una API de productos con sistema de autenticación y autorización

## Requisitos

- npm
- docker
## Installation

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose up -d mongo

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
